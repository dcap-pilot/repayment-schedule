Daily Interest Logic

  • Flat rate is converted to APR (EIR) using the Malaysia HP formula.
  • Daily rate uses actual days in year (365/366).
  • Interest per period = Opening balance × daily rate × actual days.
  • Monthly instalment uses standard amortization with APR / 12.
  • Principal = Instalment - Interest; last instalment clears remaining balance.

Instalments are rounded up to 0 decimals.

How this generator works

  1. The generator assumes monthly instalments but recognizes profit daily.
  2. Each period uses actual day count between the previous due date and the next due date.
  3. Interest is computed on reducing balance using daily rate; principal is the remainder.
  4. Final instalment is adjusted to fully settle outstanding balance.
